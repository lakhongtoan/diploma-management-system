# Decentralized Diploma Management System

## Project Title

Decentralized Diploma Management System

## Project Description

The Decentralized Diploma Management System is a blockchain-based platform built on the Stellar Soroban smart contract framework. It enables educational institutions to securely issue, store, and verify academic diplomas on-chain.

The system ensures that diploma records are immutable, transparent, and tamper-proof. Universities and educational organizations can issue digital diplomas, while employers and third parties can instantly verify their authenticity without relying on centralized databases.

By leveraging the Stellar blockchain, the platform enhances trust, reduces fraud, and simplifies the diploma verification process.

---

## Project Vision

The vision of this project is to create a trusted and decentralized ecosystem for academic credential management. Traditional diploma verification often involves manual processes, document forgery risks, and delays.

This platform aims to:

* Eliminate diploma fraud.
* Provide instant diploma verification worldwide.
* Increase transparency and trust between educational institutions, graduates, and employers.
* Build a secure and permanent academic credential infrastructure on the Stellar blockchain.

---

## Key Features

### Diploma Issuance

Authorized educational institutions can issue digital diplomas and record them permanently on-chain.

### Diploma Verification

Anyone can verify the authenticity of a diploma using its unique diploma ID.

### Immutable Records

Once issued, diploma information cannot be altered, ensuring data integrity and trustworthiness.

### Institution Authentication

Only authorized issuers can create diploma records.

### Public Transparency

Diploma status and verification data can be publicly accessed while maintaining security.

### Fraud Prevention

Blockchain-based storage prevents diploma tampering and unauthorized modifications.

### On-Chain Auditability

All diploma-related activities are recorded on-chain for complete traceability.

---

## Usage Instructions

### 1. Deploy Contract

Deploy the Soroban smart contract to the Stellar network.

### 2. Register Institution

Assign authorized institution accounts capable of issuing diplomas.

### 3. Issue Diploma

The institution submits:

* Diploma ID
* Student Name
* Student ID
* Major
* Grade Classification
* Issue Date
* Diploma File Hash

The diploma is then permanently stored on-chain.

### 4. Verify Diploma

Employers, organizations, or individuals can query a diploma using its unique Diploma ID.

### 5. Manage Records

Authorized institutions can manage diploma issuance according to system policies.
